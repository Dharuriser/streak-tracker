# ルーティン判定ルール

この文書は、アプリが表示する手順と安全調整の基準です。記録はユーザーを評価するためではなく、実際に行ったケアへ記録を合わせるために使います。

## 時間帯と記録日

- 朝：04:00〜15:59
- 夜：16:00〜翌03:59
- 00:00〜03:59の夜ケアは前日分として記録します。
- 記録日と朝／夜を切り替えることで、過去の記録を「やった」「やってない」に補正できます。

## 朝

毎日、次の順番で表示します。

1. 洗顔
2. MISSHA Vita C Plus Toner
3. Ceramedx Soothing Facial Lotion
4. Badger Kids Mineral Sunscreen SPF40

朝の日焼け止めは常に表示します。Thayers、Advanced、RoC Retinolは朝には表示しません。

## 夜・1か月目

| 曜日 | 手順 |
|---|---|
| 月 | 洗顔 → Ceramedx |
| 火 | 洗顔 → Thayers → Ceramedx |
| 水 | 洗顔 → Ceramedx |
| 木 | 洗顔 → Ceramedx → RoC Retinol少量 → 必要ならCeramedx追加 |
| 金 | 洗顔 → Ceramedx |
| 土 | 洗顔 → Ceramedx |
| 日 | 肌状態が「問題なし」なら 洗顔 → Thayers → Ceramedx。そうでなければ 洗顔 → Ceramedx |

## 2か月目以降

- 初回利用日から31日目以降を2か月目として扱います。
- 直近7日間に乾燥・赤み・ヒリつき・皮むけ・日焼けの記録がない場合、火曜のThayersをAdvancedに置き換えます。
- Advanced使用後2日以内に赤み・ヒリつき・皮むけが記録された場合、使用日から14日間Advancedを停止します。

## 安全調整

- AHAとレチノールは同じ夜に表示しません。
- ThayersとAdvancedは同じ夜に表示しません。
- 乾燥・赤み・ヒリつき・皮むけがある日と、その後3日間は、夜を「洗顔 → Ceramedx」にします。
- 日焼けを記録した当日と翌日は、夜を「洗顔 → Ceramedx」にします。
- AHAの日はレチノールを使わない注意文を表示します。
- レチノールの日はAHAを使わず、少量から使用する注意文を表示します。
- その時間帯に使わない刺激成分も画面に表示します。

強い痛み・腫れ・症状の悪化がある場合は使用を中止し、医療機関へ相談してください。

## 集計

- 合計実行回数：完了した朝・夜の合計
- 今週の実行数：月曜から現在までに完了した朝・夜の合計
- 今月の実行数：当月に完了した朝・夜の合計
- 直近の連続日数：最後に朝・夜の両方が完了した日から、過去へ連続している日数

最後の項目は参考値です。直近に未記録の日があっても、過去の連続記録を0にはしません。
